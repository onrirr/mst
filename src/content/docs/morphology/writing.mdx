The Orkhon Script is a script used to write the [Orkhon Inscriptions](https://tonga.onrir.dev/writings/orhun/) and the [Yenisean Inscriptions](https://tonga.onrir.dev/writings/yenisey/).

The original script is written left to write and is in form of a syllabary. Some letters represent the words those letters sound out. Like this letter, "ok" (meaing arrow) being a literal arrow <kbd>𐰸</kbd> or this letter reading "at" (meaning to shoot/throw) depicts an arrow being shot <kbd>𐱃</kbd>. While however cool this may be it is not a unified approach as a single letter can be read as a lot of things. To continue with our examples, the letter <kbd>𐰚</kbd> (ek) could be any of the following; ek, ke, ik, ki. 

While I believe the script itself can be considered cool, I see it of no benefit to include this sort of ambiguity in a modern script. So I have decided to make a few changes to the script.

## Modern Turkic Script

The modern Turkic script is written left to right to fit the modernity of our era better and to not run in to technical issues like the ones I did while preparing this website. The script is also written in a way that each letter represents a single sound. This way the script is more of an alphabet than a semi-syllabary which it used to be.

The writing <kbd>𐱅𐰇𐰼𐰜</kbd> now represents a clear one letter at a time system sounding out "TÜRK"

However this arises a new challenge. The letters for some sounds are the same and some non-existant.

### Vowels

For the vowel sounds, while none were missing, some were sharing the same letter. <kbd>𐰀</kbd> : a, ä <kbd>𐰃</kbd> : ı, i<kbd>𐰆</kbd> : o, u <kbd>𐰇</kbd> : ö, ü. Sınce I wanted to prevent adding new/made up letters as much as possible I fixated on using the Yeniseian variants for some of these sounds.
    
Sadly, Yenisean does not have a different symbol for <kbd>𐰆</kbd> marking both o and u so instead I reversed it, found a unicode chracter that looked like it being reversed which was <kbd>⧼</kbd> (	
Left-pointing curved angle bracket) and use that in its stead.
